# سیستم تماس / کال‌سنتر آهن‌تایم — راهنمای راه‌اندازی

خودِ **خط تلفن** (که مشتری به آن زنگ بزند یا کارشناس با آن تماس بگیرد) نیازمند
قرارداد با یک ارائه‌دهندهٔ تلفن ابری / VoIP ایرانی است و در کد قابل ساخت نیست —
باید شماره و اشتراک تهیه شود. آنچه در سایت/پنل ساخته شد در پایین آمده و پس از آن
پیشنهاد ارائه‌دهنده.

## آنچه در پنل و سایت پیاده شد (بدون نیاز به سرویس خارجی)
- **ویجت «درخواست تماس رایگان»** روی همهٔ صفحات سایت: مشتری نام + موبایل می‌گذارد
  و درخواست مستقیم به صندوق ورودی پنل (پیام‌ها) می‌رود.
- **زنگ خوردن پنل**: هر ~۲۰ ثانیه پنل ادمین تعداد سرنخ/پیام جدید را چک می‌کند؛
  با ورود مورد جدید، **صدای زنگ** پخش می‌شود و **نوتیفیکیشن دسکتاپ** روی
  لپ‌تاپ کارشناس بالا می‌آید (حتی اگر تب دیگری باز باشد).
- **کلیک-برای-تماس**: شمارهٔ هر مشتری در «میز کار من» لینک `tel:` است؛ با کلیک،
  از طریق نرم‌افزار تلفن/سافت‌فون سیستم‌عامل شماره‌گیری می‌شود.
- **پیامک خودکار**: خوش‌آمد، یادآوری اعتبار پیش‌فاکتور، یادآوری تماس به کارشناس.

## گام بعدی که به شما نیاز دارد: خط تلفن ابری
برای «زنگ زدن مشتری به یک شمارهٔ ثابت مثل کال‌سنترها» و «تماس خروجی کارشناس از
همان شماره»، یکی از این ارائه‌دهنده‌های تلفن ابری ایران را انتخاب و اشتراک بگیرید.
هر کدام مرکز تماس ابری (IVR، صف تماس، چند داخلی، ضبط مکالمه، گزارش) ارائه می‌دهند:

| ارائه‌دهنده | مناسب برای | امکانات کلیدی |
|---|---|---|
| **های‌وب (Hiweb) / تلفن ابری آسیاتک** | کسب‌وکار کوچک تا متوسط | شمارهٔ مجازی، IVR، صف، اپ موبایل/دسکتاپ |
| **الوپیک/الوویپ (AloVoIP)** | راه‌اندازی سریع | مرکز تماس ابری، کلیک‌تو‌کال، وب‌سرویس |
| **نکستل / Nexfon** | تیم فروش | داخلی‌های نامحدود، CRM-link، ضبط، گزارش |
| **کاواک / همکال (Hamcall)** | مرکز تماس حرفه‌ای | صف پیشرفته، مانیتورینگ زنده، API |
| **مخابرات — خط ثابت + پنل VoIP** | ساده‌ترین | فقط یک خط ثابت + سافت‌فون |

### توصیه
- برای شروع کم‌هزینه: یک ارائه‌دهندهٔ **تلفن ابری با API/وب‌سرویس** (مثل نکستل یا
  الوویپ) بگیرید که دو چیز بدهد: (۱) یک **شمارهٔ ثابت واحد** برای تبلیغ در سایت،
  (۲) **کلیک-تو-کال از طریق API** تا از داخل پنل، تماس خروجی هم از همان شماره برقرار
  شود (به‌جای شماره‌گیری با موبایل شخصی کارشناس).
- وقتی اشتراک گرفتید و API/شمارهٔ داخلی در اختیار داشتید، بگویید تا:
  - شمارهٔ ثابت جدید را در تنظیمات سایت (اطلاعات تماس) بگذارم،
  - دکمهٔ کلیک-تو-کال پنل را به API آن ارائه‌دهنده وصل کنم تا تماس واقعاً از خط
    شرکت برقرار شود و در CRM ثبت گردد،
  - در صورت وجود وب‌هوک تماس ورودی، «pop-up مشتری» روی پنل هنگام زنگ خوردن اضافه کنم.

## نکته
هیچ‌کدام از این ارائه‌دهنده‌ها API عمومیِ یکسان ندارند؛ پس اتصال کلیک-تو-کال به
API فقط بعد از انتخاب ارائه‌دهنده و دریافت مستندات/کلید آن ممکن است.
